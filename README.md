# boarding_money_app

A new Flutter project.

## Fast development run

Use this command for faster repeat runs:

```powershell
.\dev_run.ps1
```

It runs:

```powershell
flutter run --debug --fast-start --no-pub
```

Notes:
- First run is always slower (Gradle + Flutter cache warm-up).
- Later runs are much faster.
- If dependencies changed, run `flutter pub get` once before `dev_run.ps1`.
